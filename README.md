## Install as client

This project uses [node](http://nodejs.org) and [yarn](https://yarnpkg.com). Go check them out if you don't have them locally installed.

```sh
$ yarn
```

## Prerequisites

- yarn >=1.22.0
- node >=14.17.0

## Install

```sh
yarn
```

## Usage

```sh
yarn start
```

## Build

```sh
yarn build
```
